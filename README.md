# SARA Pay — Banking Transaction Management System

A simple yet complete banking transaction management system built with PHP. Handles deposits, withdrawals, transfers, and account management with a clean MVC architecture.

## ✨ Features

- **Account Management** — Create, update, and manage bank accounts
- **Transactions** — Deposit, withdraw, and transfer
- **Transaction History** — Full audit trail with timestamps
- **Balance Tracking** — Real-time balance updates per account

## 🛠 Tech Stack

| Layer | Technology |
|-------|-----------|
| Backend | PHP |
| Database | MySQL |
| Architecture | MVC |

## 🚀 Quick Start

```bash
git clone https://github.com/bdlogicalerror/sarapy_managmnet.git
cd sarapy_managmnet
# Edit config.php with your DB credentials
php index.php
```
